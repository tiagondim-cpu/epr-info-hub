# Questionário de Usabilidade — System Usability Scale (SUS)

> Instrumento padronizado (Brooke, 1996) para medir a usabilidade percebida do
> **EPR Info Hub**. Meta do projeto: **SUS ≥ 75** (referência: o SEI/UnB obteve 59,78).

## Como aplicar

1. Peça ao participante (estudante do EPR) que **use o app livremente por alguns
   minutos** — ou complete as tarefas-roteiro abaixo — antes de responder.
2. O participante responde **às 10 afirmações** numa escala de **1 (Discordo
   totalmente) a 5 (Concordo totalmente)**. Não há resposta "certa".
3. Aplicar com **pelo menos 5 estudantes** (recomendado para resultados estáveis).
4. Calcular a pontuação conforme a seção [Cálculo](#cálculo-da-pontuação).

### Tarefas-roteiro sugeridas (opcional, antes do questionário)

- [ ] Descobrir o prazo para trancamento automático de uma disciplina.
- [ ] Encontrar o formulário de inscrição no estágio obrigatório.
- [ ] Usar a busca para achar informações sobre o IRA.
- [ ] Localizar o e-mail da secretaria do EPR.

## Cálculo da pontuação

Para cada participante:

1. **Itens ímpares (1, 3, 5, 7, 9):** contribuição = (resposta − 1).
2. **Itens pares (2, 4, 6, 8, 10):** contribuição = (5 − resposta).
3. Some as 10 contribuições (resultado de 0 a 40).
4. **Multiplique por 2,5** → pontuação SUS individual (0 a 100).

A pontuação SUS final do produto é a **média** das pontuações individuais.

### Interpretação de referência

| Faixa SUS | Avaliação        |
| --------- | ---------------- |
| > 80,3    | Excelente (A)    |
| 68 – 80,3 | Bom (B/C)        |
| 51 – 68   | OK (D)           |
| < 51      | Ruim (F)         |

> SUS não é porcentagem — 75 não significa "75%". É uma pontuação relativa numa
> escala de 0 a 100; **68 é a média da indústria**.
